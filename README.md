# 📍 SpotSave

TikTok・Instagram・YouTubeで見つけた場所を保存するアプリです。

## 機能

- URL・店名・住所・カテゴリを保存
- カテゴリ絞り込み・キーワード検索
- お気に入り登録（⭐ 一番上に表示）
- Google Mapsで地図を開く
- 保存したURLを新しいタブで開く
- 編集・削除
- 画像アップロード
- データはlocalStorageに保存（リロードしても消えない）

## 技術スタック

- React 19
- Vite 8
- React Router v7

## ローカル起動

```bash
npm install
npm run dev
```

http://localhost:5173 をブラウザで開く

## デプロイ

Vercel にそのままデプロイできます。
